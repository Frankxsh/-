# 消息

千门万户曈曈日，总把`新桃`换旧符

## 代码演示

### 基本用法

<Playground>
  <Button 
    theme='primary' 
    outline
    onClick={
      () => Notify.open({
        message: '这是一条短消息',
        callback: () => console.log('短消息')
      })
    }
  >短消息</Button>
  <Button 
    theme='primary' 
    outline
    onClick={
      () => Notify.open({
        duration: 3000,
        message: '这是一条很长很长很长很长的消息'
      })
    }
  >长消息</Button>
</Playground>

### 自定义位置

<Playground>
  <Button 
    theme='primary' 
    outline
    onClick={
      () => Notify.open({
        message: '位置改变的消息',
        duration: 200000,
        position: {
          top: 100,
          right: 100,
        }
      })
    }
  >位置改变</Button>
</Playground>
